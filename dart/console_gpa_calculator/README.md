# Console GPA Calculator (Dart)

Interactive Dart console app to calculate GPA.

## Features

- Manual subject/grade entry
- Optional CSV import to calculate GPA from file input
- Pass/fail status per subject and overall
- CSV export with custom save path
- Styled terminal launch banner

## Run

```powershell
cd "dart/console_gpa_calculator"
dart run
```

## Input Modes

1. Manual entry
2. CSV import

CSV import format (minimum):

```csv
Subject,Grade
Math,A
Physics,78
```
